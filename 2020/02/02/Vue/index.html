  1. 点击循环切换：

    • 使用 ! ：<button v-on:click = 'isRed = !isRed'>切换</button>
    • 使用方法：
      • <button v-on:click = 'changeColor'>切换</button>
      • methods:{ changeColor(){ this.isRed = !isRed}}

  2. 单向数据流：data ==> view

  3. 只有表单控件里才能实现双向数据绑定

  4. 双向数据绑定的内部原理：v-bind:value 和 v-on:input，还有 definedProperty（渲染原理）

  5. 组件首字母大写

  6. 如果是组件，则组件中的 data 必须是函数，不能是对象，即必须使用 date(){ return {} } 或者 data : function(){ return {} }

  7. 属性名=属性值 ==> 可以简写为属性名

  8. v-bind: 可以简写成:，即v-bind:class="" ==> :class=""

  9. v-on:click="" ==> @click=""
